Agent Initialization and Cleanup

Agent programs, like environments, often need to allocate and free various data structures. The agent_init and agent_cleanup function do not take any input or return an data and are called at the beginning and end of a learning experiment, respectively. The agent_init function also receives the Task Specification string as input. The agent_init function usually parses the Task Specification and stores various information encoded in the string. For example, after parsing the Task Specification, the agent_init function can then initialize the value function array to the size state space using the number of states from the Task Specification. Remember the Task Specification is not required to contain any information (NULL or empty string).
